3 Global theory of open and ample relations

3.1 Preliminaries

3.1.1 Vector bundles operations

Definition 3.1
#

For every bundle \(p : E ‚Üí B\) and every map \(f \! :B' ‚Üí B\), the pull-back bundle \(f^*E ‚Üí B'\) is defined by \(f^*E = \{ (b', e) ‚àà B' √ó E \; |\; p(e) = f(b')\} \) with the obvious projection to \(B'\).

The case of vector bundles.

Definition 3.2
#

Let \(E ‚Üí B\) and \(F ‚Üí B\) be two vector bundles over some smooth manifold \(B\). The bundle \(\operatorname{Hom}(E, F) ‚Üí B\) is the set of linear maps from \(E_b\) to \(F_b\) for some \(b\) in \(B\), with the obvious project map.

Set-theoretically, one can define \(\operatorname{Hom}(E, F)\) as the set of subsets \(S\) of \(E √ó F\) such that there exists \(b\) such that \(S ‚äÇ E_b √ó F_b\) and \(S\) is the graph of a linear map. But the type theory formalization will use other tricks here. The facts that really matter are listed in lemma 3.5.

3.1.2 Jets spaces

Definition 3.3
#

Uses

  • Definition 3.1
  • Definition 3.2

Let \(M\) and \(N\) be smooth manifolds. Denote by \(p_1\) and \(p_2\) the projections of \(M √ó N\) to \(M\) and \(N\) respectively.

The space \(J^1(M, N)\) of \(1\)-jets of maps from \(M\) to \(N\) is \(Hom(p_1^*TM, p_2^*TN)\)

We will use notations like \((m, n, œÜ)\) to denote an element of \(J^1(M, N)\), but one should keep in mind that \(J^1(M, N)\) is not a product, since \(œÜ\) lives in \(\operatorname{Hom}(T_mM, T_nN)\) which depends on \(m\) and \(n\).

Definition 3.4
#

Uses

  • Definition 3.3

The \(1\)-jet of a smooth map \(f \! :M ‚Üí N\) is the map from \(m\) to \(J^1(M, N)\) defined by \(j^1f(m) = (m, f(m), T_mf)\).

The composition of a section \(\mathcal{F}\! :M ‚Üí J^1(M, N)\) with the projection onto \(N\) will sometimes be denoted by \(\operatorname{bs}\mathcal{F}\! :M ‚Üí N\) and called the base map of \(\mathcal{F}\).

Lemma 3.5
#

Uses

For every smooth map \(f \! :M ‚Üí N\),

  1. \(j^1f\) is smooth

  2. \(j^1f\) is a section of \(J^1(M, N) ‚Üí M\)

  3. \(j^1f\) composed with \(J^1(M, N) ‚Üí N\) is \(f\).

Proof ‚ñº

This is obvious by construction‚Ä¶

Definition 3.6
#

Uses

  • Definition 3.3
  • Lemma 3.5

A section \(\mathcal{F}\) of \(J^1(M, N) ‚Üí M\) is called holonomic if it is the \(1\)‚Äìjet of its base map. Equivalently, \(\mathcal{F}\) is holonomic if there exists \(f \! :M ‚Üí N\) such that \(\mathcal{F}= j^1f\), since such a map is necessarily \(\operatorname{bs}\mathcal{F}\).

3.2 First order differential relations

Definition 3.7
#

Uses

  • Definition 3.3

A first order differential relation for maps from \(M\) to \(N\) is a subset \(\mathcal{R}\) of \(J^1(M, N)\).

Definition 3.8
#

Uses

  • Definition 3.7
  • Definition 3.4

A formal solution of a differential relation \(\mathcal{R}‚äÇ J^1(M, N)\) is a section of \(J^1(M, N) ‚Üí M\) taking values in \(\mathcal{R}\). A solution of \(\mathcal{R}\) is a map from \(M\) to \(N\) whose \(1\)‚Äìjet extension is a formal solution.

Definition 3.9
#

Uses

  • Definition 3.8

A homotopy of formal solutions of \(\mathcal{R}\) is a family of sections \(\mathcal{F}: ‚Ñù √ó M ‚Üí J^1(M, N)\) which is smooth over \([0, 1] √ó M\) and such that each \(m ‚Ü¶ \mathcal{F}(t, m)\) is a formal solution when \(t\) is in \([0, 1]\).

Definition 3.10
#

Uses

  • Definition 3.8
  • Definition 3.9

A first order differential relation \(\mathcal{R}‚äÇ J^1(M, N)\) satisfies the \(h\)-principle if every formal solution of \(\mathcal{R}\) is homotopic to a holonomic one. It satisfies the parametric \(h\)-principle if, for every manifold with boundary \(P\), every family \(\mathcal{F}: P √ó M ‚Üí J^1(M, N)\) of formal solutions which are holonomic for \(p\) in \(ùìù(‚àÇP)\) is homotopic to a family of holonomic ones relative to \(ùìù(‚àÇP)\). It satisfies the parametric \(h\)-principle if, for every manifold with boundary \(P\), every family \(\mathcal{F}: P √ó M ‚Üí J^1(M, N)\) of formal solutions is homotopic to a family of holonomic ones.

3.3 The \(h\)-principle for open and ample differential relations

In this chapter, \(X\) and \(Y\) are smooth manifolds and \(\mathcal{R}\) is a first order differential relation on maps from \(X\) to \(Y\): \(\mathcal{R}‚äÇ J^1(X, Y)\). For any \(œÉ = (x, y, œÜ)\) in \(\mathcal{R}\) and any dual pair \((Œª, v) ‚àà T^*_xV √ó T_xV\), we set:

\[ \mathcal{R}_{œÉ, Œª, v} = \operatorname{Conn}_{œÜ(v)}\left\{ w ‚àà T_yY \; ;\; \big (x,\; y,\; œÜ + (w - œÜ(v))‚äóŒª\big ) ‚àà \mathcal{R}\right\} \]

where \(\operatorname{Conn}_a A\) is the connected component of \(A\) containing \(a\). In order to decipher this definition, it suffices to notice that \(œÜ + (w - œÜ(v))‚äóŒª\) is the unique linear map from \(T_xX\) to \(T_yY\) which coincides with \(œÜ\) on \(\ker Œª\) and sends \(v\) to \(w\). In particular, \(w = œÜ(v)\) gives back \(œÜ\).

Of course we will want to deal with more that one point, so we will consider a vector field \(V\) and a \(1\)‚Äìform \(Œª\) such that \(Œª(V) = 1\) on some subset \(U\) of \(X\), a formal solution \(F\) (defined at least on \(U\)), and get the corresponding \(\mathcal{R}_{F, Œª, v}\) over \(U\).

One easily checks that \(\mathcal{R}_{œÉ, Œ∫^{-1}Œª, Œ∫v} = Œ∫\mathcal{R}_{œÉ, Œª, v}\) hence the above definition only depends on \(\ker Œª\) and the direction \(‚ÑùV\).

Lemma 3.18
#

Uses

  • Definition 3.14

If \(\mathcal{R}\) is ample then, for any parameter space \(P\), \(\mathcal{R}^P\) is also ample.

Proof ‚ñº

We fix \(œÉ = (x, y, œà)\) in \(\mathcal{R}^P\). For any \(Œª = (Œª_X, Œª_P) ‚àà T^*_xX √ó T^*_pP\) and \(v = (v_X, v_P) ‚àà T_xX √ó T_pP\) such that \(Œª(v) = 1\), we need to prove that \(\operatorname{Conv}\mathcal{R}^P_{œÉ, Œª, v} = T_yY\). Unfolding the definitions gives:

\[ \mathcal{R}^P_{œÉ, Œª, v} = \operatorname{Conn}_{œÜ(v)}\left\{ w ‚àà T_yY \; ;\; \big (x,\; y,\; œà ‚àò Œπ_{x, p} + (w - œà(v))‚äóŒª_X\big ) ‚àà \mathcal{R}\right\} . \]

A degenerate but easy case is when \(Œª_X = 0\). Then the condition on \(w\) becomes \(œà ‚àò Œπ_{x, p} ‚àà \mathcal{R}\), which is true by definition of \(\mathcal{R}^P\), so \(\mathcal{R}^P_{œÉ, Œª, v} = T_yY\).

We now assume \(Œª_X\) is not zero and choose \(u ‚àà T_xX\) such that \(Œª_X(u) = 1\). We then have \(\mathcal{R}^P_{œÉ, Œª, v} = \mathcal{R}_{Œ®œÉ, Œª_X, u} + œà(v) - œà‚àòŒπ_{x, p}(u)\). Because \(\mathcal{R}\) is ample and taking convex hull commutes with translation, we get that \(\operatorname{Conv}\mathcal{R}^P_{œÉ, Œª, v} = T_yY\).

Proof of Theorem 3.17 ‚ñº

Lemmas 3.13 and 3.18 prove we can assume there are no parameters. So we start with a single formal solution \(F\) of \(\mathcal{R}\), which is holonomic near some closed subset \(A ‚äÇ X\).

We first assume \(X\) is closed, and will then explain the proof adjustments needed in the non-compact case. By compactness, there are finite many compact subsets \((K_i)_{1 ‚â§ i ‚â§ N}\) contained in coordinate charts \(U_i\) and such that \(\bigcup K_i = X\).

We prove by induction on \(i\) from \(0\) to \(N\) that there are formal solutions \(F_i\), starting with \(F_0 = F\), that are homotopic to \(F\) relative to \(A\), holonomic on \(K_{‚â§ i} := \bigcup _{j ‚â§ i} K_j\) and whose base maps are \((1-2^{-i})Œµ\)-close to that of \(F\) (this contrived bound will be convenient for the non-compact case).

Assume \(F_i\) has been constructed for some \(i {\lt} N\). We now want to construct it on \(U_{i+1}\). Lemma 3.15 ensures the pull-back of \(\mathcal{R}\) in the chart corresponding to \(U_{i+1}\) is ample in the sense of the preceding chapter. Hence Lemma 2.16 can be applied to construct \(F_{i+1}\), using the image of \(K_{i+1}\) as \(K_0\) and the image of \(A ‚à© K_{i+1} ‚à© K_{‚â§ i}\) as \(C\). It is holonomic on \(K_{‚â§ i+1}\) because it agrees with \(F_i\) near \(K_{‚â§ i}\) and is holonomic near \(K_{i+1}\).

Once all \(F_i\) are constructed, we define \(F^t\) to be the concatenation of all homotopies relating \(F_i\) to \(F_{i+1}\).

If \(X\) is not compact, then one can use a countable family of subsets \((U_i, K_i)\) which is locally finite (ie. every point \(x\) has a neighborhood intersecting only finitely many \(U_i\)). The way we have chosen \(C^0\)-bounds ensures that each \(\operatorname{bs}F_i\) is still at distance at most \(Œµ\) from \(\operatorname{bs}F\). We now have countably many homotopies to concatenate, so we need to reparametrize the \(i\)-th homotopy by an interval of length \(2^{-i}\). This give a family of sections of \(J^1(X, Y)\) parametrized by \(t ‚àà [0, 1)\). But our local finiteness assumption implies that, for each each \(x\), there is some \(t_0 {\lt} 1\) and some neighborhood \(U\) of \(x\) such that our family is \(t\)‚Äìindependent on \(U\) for \(t ‚â• t_0\). So we can extend to \(t = 1\). The resulting family is smooth since smoothness is a local condition in both \(x\) and \(t\).

Theorem 3.19 Smale 1958
#

There is a homotopy of immersions of \(ùïä^2\) into \(‚Ñù^3\) from the inclusion map to the antipodal map \(a : q ‚Ü¶ -q\).

Proof ‚ñº

We denote by \(Œπ\) the inclusion of \(ùïä^2\) into \(‚Ñù^3\). We set \(j_t = (1-t)Œπ + ta\). This is a homotopy from \(Œπ\) to \(a\) (but not animmersion for \(t=1/2\)). Using the canonical trivialization of the tangent bundle of \(‚Ñù^3\), we can set, for \((q, v) ‚àà Tùïä^2\), \(G_t(q, v) = \mathrm{Rot}_{Oq}^{œÄt}(v)\), the rotation around axis \(Oq\) with angle \(œÄt\). The family \(œÉ : t ‚Ü¶ (j_t, G_t)\) is a homotopy of formal immersions relating \(j^1Œπ\) to \(j^1a\). It is homotopic by reparametrization to a homotopy of formal immersions relating \(j^1Œπ\) to \(j^1a\) which are holonomic for \(t\) near the \(0\) and \(1\).

The above theorem ensures this family is homotopic, relative to \(t = 0\) and \(t = 1\), to a family of holonomic formal immersions, ie a family \(t ‚Ü¶ j^1f_t\) with \(f_0 = Œπ\), \(f_1 = a\), and each \(f_t\) is an immersion.
